当前代码放入缓存区做准备:  git add ./

提交到本地并注释(wu):     git commit -m wu

推送到远程服务器:         git push origin yangzhou

切换分支:                git checkout yangzhou

切换之后更新一下分支:      git pull origin yangzhou

注意：切换分支前先提交一下
